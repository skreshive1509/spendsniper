SpendSniper

Modern Android Expense Tracking

Release Overview

SpendSniper v2.0 is a polished Android expense tracking app that makes personal finance management effortless. Add expenses with title, amount, category, and auto-filled date; organize spending into custom categories; and access intelligent dashboards, reports, export tools, and cloud sync.

Technology Stack

Built with modern, robust technology for a fast, secure, and intuitive user experience.

Kotlin + Java - Official Android dev languages
MVVM Architecture - ViewModel & LiveData
Firebase - Auth, Firestore, Storage
Material Design - Modern UI components
MPAndroidChart - Dynamic visualizations
PDF & CSV Export - Data portability

Current Status

SpendSniper v2.0 is now publicly released with Firebase authentication, real-time cloud sync, PDF/CSV export, detailed reports, search, and a polished Material UI.

What’s Included in v2.0

  • Secure Firebase Authentication with email registration and login.
  • Integrated password recovery with email reset flow.
  • Real-time cloud synchronization for expenses and user profiles.
  • Advanced monthly reporting with pie chart category breakdown and bar chart comparison.
  • Global search by title, category, and date with instant results.
  • Export full expense history as PDF and CSV for printing or analysis.
  • Improved Android UI with splash screen, FAB, RecyclerView, and Material design polish.
  • Versioned release info, better performance, and stable Kotlin/Flow architecture.

v1.0 vs v2.0 Comparison

SpendSniper v1.0
  • Local expense management only
  • Basic category organization
  • Simple dashboard overview
  • No cloud backup or sync
  • No export or recovery options
SpendSniper v2.0
  • Secure Firebase login and user profiles
  • Real-time cloud sync across devices
  • PDF and CSV export for sharing and analysis
  • Advanced monthly reports and charts
  • Global search and password recovery
