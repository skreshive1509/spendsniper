Application Overview

SpendSniper is a modern and intuitive Android application designed for effortless personal expense tracking. It empowers users to monitor their spending habits through a clean, user-friendly interface, powerful reporting tools, and robust search capabilities.

Key Features

1. Core Expense Management

  • Add Expenses: Quickly add new expenses with title, amount, and category. Date auto-set to today.
  • Categorization: Organize expenses into user-defined categories like "Groceries," "Fast Food," etc.
  • Central Dashboard: Access all primary functions from one screen.

2. Advanced Monthly Reporting

  • Date Filtering: Select month and year to view history.
  • Pie Chart Visualization: Dynamic chart with category breakdown and percentages.
  • Side-by-Side Comparison: FAB toggles to bar chart comparing current and previous month.
  • Itemized Totals: Scrollable list of exact category totals.

3. Global Search Functionality

  • Context-Aware UI: Search by name, category, or date.
  • Instant Results: Real-time filtering as you type.

4. Data Export and Portability

  • Download Reports: Export full history with one tap.
  • Multiple Formats: PDF and CSV (Excel) supported.

5. Polished User Experience (UI/UX)

  • Immersive Welcome Screen: Full-screen splash with slogan.
  • Dynamic Titles: Each screen shows module and version.
  • Modern Design: FAB, RecyclerView, clean layout.
  • Responsive Performance: Powered by Kotlin Coroutines and Flow.
